## Description
A web-based expense tracker application built with React, Ant Design, and Lucide Icons. This app helps users manage their income and expenses, view monthly summaries, and track transactions in real-time.

## Features
- Add, edit, and delete transactions.
- View total balance, monthly income, and monthly expenses.
- Filter transactions by type (income/expense) and date range.
- Dark mode support for better user experience.
- Responsive design for mobile and desktop.

## Technologies Used
- **Frontend**: React, Ant Design, Lucide Icons
- **Backend**: Node.js, Express, MongoDB (if applicable)
- **Other Tools**: Axios, React Hot Toast, Day.js
